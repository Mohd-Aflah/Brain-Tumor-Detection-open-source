# Contributing to Brain Tumor Detection System

Thank you for your interest in contributing to this open-source project! 

This project is created by **mohd-aflah(github)** primarily for educational purposes, college projects, and research. 

## How You Can Contribute

1. **Bug Reports & Feature Requests**: Feel free to open an issue if you encounter a bug or have a suggestion for a new feature.
2. **Code Contributions**: 
   - Fork the repository.
   - Create a new branch for your feature or bug fix (`git checkout -b feature-name`).
   - Make your changes and commit them (`git commit -m "Add some feature"`).
   - Push to the branch (`git push origin feature-name`).
   - Open a Pull Request.

## Important Notes

* **NOT FOR MEDICAL USE**: Please remember that this tool is not a certified medical device. Do not submit features that claim clinical accuracy or attempt to provide medical diagnosis.
* Keep the code clean, fast, and optimized.
* Update documentation (like `README.md` or comments) if you make significant changes.

Thank you for helping make this project better for everyone!
