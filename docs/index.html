About The Project

This project was developed to bridge the gap between complex deep learning models and accessible user interfaces for medical image analysis.

It provides a complete pipeline from image upload to YOLO-based semantic segmentation, concluding with analytical assessment. It is perfectly suited for college curriculum, bioinformatics research, and AI enthusiasts.

  • Python & Flask
  • Ultralytics YOLO (PyTorch)
  • OpenCV & NumPy
  • HTML5, CSS3, JS
